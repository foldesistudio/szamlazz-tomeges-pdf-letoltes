# szamlazz.hu - Tömeges PDF letöltés (userscript)
Tampermonkey userscript a szamlazz.hu kimenő számlalistához.

„Ingyen letöltés” gombot ad az oldalhoz, ami a kijelölt számlákat/bizonylatokat automatikusan, egymás után letölti tömegesen PDF-ben (#free csomagnál is).

## Telepítés ## 
1. Telepíts egy userscript-kezelőt (pl. Tampermonkey)

2. Nyisd meg a script URL-jét (raw `szamlazz-tomeges-pdf-letoltes.user.js`), majd Telepítés

3. Frissíts rá a szamlazz.hu kimenő számlalistára.

## Használat ## 
1. Menj a Kimenő számlalistára

2. Jelöld ki a letölteni kívánt bizonylatokat

3. Kattints a frissen megjelenő „Ingyen letöltés” gombra középen

4. Várd meg, amíg sorban lefut (közben ne nagyon kattintgass)

## Fontos megjegyzések ##
A userscript fejlesztője semmilyen garanciát és/vagy felelősséget nem vállal.
Ez egy nem hivatalos userscript, teljes mértékben hobbi projekt.
